Edge-Based Standing-Water Detection via FSM-Guided Tiering and Multi-Model Consensus

Oliver Larsen, Mahyar T. Moghaddam

IEEE ICSA 2026 · Software Architecture in Practice Track

Accepted Edge AI / IoT
Code

Standing water detection on agricultural edge devices using a finite-state-machine (FSM) to guide intelligent tiering between lightweight and heavyweight ML models. The system adapts inference strategy based on device resources and environmental confidence, achieving reliable detection on resource-constrained hardware through multi-model consensus.

TL;DR

We built an edge AI system that detects standing water on farms using a state-machine to intelligently switch between cheap and expensive ML models depending on device resources.

RAD-AI: Rethinking Architecture Documentation for AI-Augmented Ecosystems

Oliver Aleksander Larsen, Mahyar T. Moghaddam

IEEE ICSA 2026 · ANGE 2026 Workshop

Accepted AI Governance
Code

We propose backward-compatible extensions of arc42 (8 sections) and C4 (3 diagram types) to handle AI-specific architectural concerns including data provenance, model lifecycle, drift monitoring, and explainability. The extended framework improves EU AI Act compliance coverage from approximately 36% to approximately 93%, providing practitioners with a structured approach to document AI-augmented systems.

TL;DR

Extended arc42 and C4 to handle AI systems, improving EU AI Act compliance from ~36% to ~93%.

BitSov: A Composable Bitcoin-Native Architecture for Sovereign Internet Infrastructure

Oliver Aleksander Larsen, Rasmus Thorsen Larsen, Mahyar T. Moghaddam

IEEE ICSA 2026 · BlockArch 2026 Workshop

Accepted Blockchain

Eight-layer composable framework for Bitcoin-native sovereign internet infrastructure. The architecture leverages Bitcoin and Lightning Network protocols to provide censorship-resistant networking, identity, storage, and communication layers that operate without dependence on traditional centralized infrastructure.

TL;DR

Eight-layer architecture using Bitcoin and Lightning for censorship-resistant internet.

SAGAI-MID: A Generative AI-Driven Middleware for Dynamic Runtime Interoperability

Oliver Aleksander Larsen, Mahyar T. Moghaddam

IEEE ICSA 2026 · SAGAI 2026 Workshop

Accepted AI Middleware
Code

FastAPI middleware using LLMs to resolve schema mismatches between heterogeneous APIs at runtime. The system intercepts requests, detects structural incompatibilities, and dynamically transforms payloads to achieve interoperability without manual adapter code, reaching approximately 90% accuracy on schema mismatch resolution.

TL;DR

Middleware using LLMs to fix API schema mismatches at runtime — 90% accuracy.
