Edge-Based Standing-Water Detection via FSM-Guided Tiering and Multi-Model Consensus

Authors: Oliver Larsen, Mahyar T. Moghaddam

Venue: IEEE ICSA 2026, Software Architecture in Practice Track

Abstract

Standing water detection on agricultural edge devices (Raspberry Pi + Jetson). We present an FSM-guided control plane that selects between local and offloaded inference tiers based on resource availability and environmental conditions. A multi-model YOLO ensemble combined with diurnal sensor fusion achieves robust detection across varying lighting and weather conditions, enabling real-time flood monitoring on resource-constrained hardware.

TL;DR

We built an edge AI system that detects standing water on farms using a state-machine to intelligently switch between cheap and expensive ML models depending on device resources.

Keywords

edge computing, FSM orchestration, model tiering, multi-model consensus, sensor fusion

Frequently Asked Questions

What hardware does this system run on?

The system runs on a Raspberry Pi for lightweight inference and offloads to a Jetson AGX Orin for heavier models, connected via MQTT.

How does the FSM control model selection?

A finite-state machine monitors resource utilization and environmental signals to transition between inference tiers — from a fast, low-power model to a full YOLO ensemble when conditions demand it.

What accuracy does the system achieve?

The multi-model consensus approach with diurnal sensor fusion achieves robust detection across varying lighting conditions, outperforming single-model baselines on our agricultural dataset.
