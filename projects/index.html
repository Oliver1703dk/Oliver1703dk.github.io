Projects

CarboPlanner

Status: Active

Sports nutrition startup. Co-founded and raised 1,090,000 DKK. Built a full-stack platform with React, Flutter, NestJS, and MongoDB for personalized macro tracking and meal planning.

Tags: React, Flutter, NestJS, MongoDB, Startup

Hybrid LLM Meal Recommender

Status: Completed

Hybrid LLM recommender integrated into CarboPlanner. Combines in-context learning with LoRA/PEFT fine-tuned Mistral-7B for personalized athlete meal planning. 46.7% accuracy, 100% format adherence. Under revision at Results in Engineering.

Tags: LLM, LoRA, Mistral-7B, Flask, CarboPlanner

Flood Detection System

Status: Active

Edge architecture for standing-water detection in agricultural fields. Raspberry Pi + Jetson AGX Orin running YOLO models with FSM-based orchestration over MQTT.

Tags: Edge AI, YOLO, Raspberry Pi, Jetson, MQTT

Stock Analysis System

Status: Completed

Multi-agent LLM trading system with autonomous analysis pipelines. Built with Next.js frontend, Flask backend, MongoDB, Celery task queue, and Redis.

Tags: Multi-agent, LLM, Next.js, Flask
