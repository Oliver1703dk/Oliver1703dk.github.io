Oliver Aleksander Larsen

Oliver Aleksander Larsen is a software architecture researcher at the University of Southern Denmark (SDU), studying how architecture adapts when AI becomes both a design tool and a design subject, from resource-constrained edge systems to regulatory compliance.

MSc Software Engineering, supervised by Mahyar T. Moghaddam.

Projects

  • CarboPlanner - Sports nutrition startup. Co-founded and raised 1,090,000 DKK. Built a full-stack platform with React, Flutter, NestJS, and MongoDB for personalized macro tracking and meal planning.
  • Hybrid LLM Meal Recommender - Hybrid LLM recommender integrated into CarboPlanner. Combines in-context learning with LoRA/PEFT fine-tuned Mistral-7B for personalized athlete meal planning. 46.7% accuracy, 100% format adherence. Under revision at Results in Engineering.
  • Flood Detection System - Edge architecture for standing-water detection in agricultural fields. Raspberry Pi + Jetson AGX Orin running YOLO models with FSM-based orchestration over MQTT.
  • Stock Analysis System - Multi-agent LLM trading system with autonomous analysis pipelines. Built with Next.js frontend, Flask backend, MongoDB, Celery task queue, and Redis.
